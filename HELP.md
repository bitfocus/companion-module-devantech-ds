## Devantech dS

This module controls Devantech dS relay boards via TCP.

## Usage

Upon launch enter the IP of the dS board you wish to control.

### Available variables

- Not implemented yet

### Available actions

- Set relay status (on or off)
- Set I/O status (on or off for outputs)

### Available feedbacks

- Not implemented yet
